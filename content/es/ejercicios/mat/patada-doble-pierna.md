La Patada Doble de Pierna combina una patada dinámica con extensión de espalda, trabajando todo el cuerpo posterior mientras estira el frente. Este ejercicio clásico desarrolla fuerza de cadena posterior.

## Patea y Extiende

Este ejercicio usa el impulso de las patadas para ayudar a propulsar hacia la extensión, creando un patrón de movimiento fluido.

## Beneficios

- **Fuerza de Espalda**: Desarrolla extensores espinales
- **Apertura de Hombros**: Estira pecho y hombros
- **Trabajo de Isquiotibiales**: Activa isquiotibiales en la patada
- **Coordinación**: Sincroniza patada con extensión

## Errores Comunes a Evitar

1. **Patear muy fuerte** - Mantén patadas controladas
2. **Comprimir cuello** - Mantén cuello largo en extensión
3. **Apresurarse** - Permite tiempo para extensión completa
4. **Olvidar respiración** - Exhala en extensión

## Consejos para el Éxito

- Usa las patadas para generar impulso para extensión
- Mantén tu hueso púbico en el mat durante todo
- Alcanza tus manos hacia tus pies en extensión
