Rodar Atrás es un ejercicio fundamental que enseña la curva C y construye control abdominal. Esta es una gran preparación para ejercicios de rodar más avanzados.

## Controla la Curva

Este ejercicio desarrolla la habilidad de mantener una columna redondeada mientras te mueves, construyendo la base para roll ups y otros ejercicios.

## Beneficios

- **Control del Core**: Desarrolla fuerza abdominal
- **Articulación Espinal**: Enseña curva C
- **Fundamento**: Prepara para rodar avanzado
- **Conciencia**: Construye conexión mente-cuerpo

## Errores Comunes a Evitar

1. **Espalda plana** - Mantén columna redondeada en curva C
2. **Ir muy lejos** - Detente en espalda baja
3. **Usar impulso** - Rueda con control
4. **Contener respiración** - Exhala mientras ruedas hacia atrás

## Consejos para el Éxito

- Piensa en recoger tu ombligo hacia tu columna
- Mantén la curva en tu columna durante todo
- Usa tu exhalación para iniciar el rodar
