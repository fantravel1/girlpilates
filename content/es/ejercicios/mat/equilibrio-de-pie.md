El Equilibrio de Pie es un ejercicio fundamental que desafía tu sistema de estabilidad. El buen equilibrio es esencial para actividades diarias y prevención de lesiones.

## Encuentra Tu Centro

El equilibrio requiere la integración de múltiples sistemas corporales - visual, vestibular y propioceptivo. Entrenar el equilibrio mejora todos estos.

## Beneficios

- **Mejora de Equilibrio**: Entrena directamente la estabilidad
- **Fuerza de Tobillo**: La pierna de apoyo trabaja duro para estabilizar
- **Activación del Core**: El core se activa para mantener postura erguida
- **Prevención de Caídas**: Mejor equilibrio reduce riesgo de caídas

## Errores Comunes a Evitar

1. **Contener respiración** - Respira naturalmente durante todo
2. **Tensarse** - Quédate relajado, permite pequeños ajustes
3. **Mirar abajo** - Mantén mirada al frente
4. **Rendirse muy rápido** - El equilibrio mejora con práctica

## Consejos para el Éxito

- Activa tu core para mejor estabilidad
- Presiona firmemente a través de tu pie de apoyo
- Empieza con apoyo y progresa a estar de pie libre
