Cisne Buceo Preparación introduce el componente de mecimiento del Cisne Buceo de manera controlada, construyendo confianza y fuerza de espalda.

## Mece con Control

Este ejercicio enseña el movimiento de mecimiento del cisne buceo mientras permite que las manos ayuden para seguridad.

## Beneficios

- **Extensión de Espalda**: Fortalece extensores espinales
- **Patrón de Mecimiento**: Introduce movimiento de cisne buceo
- **Confianza**: Construye comodidad con el movimiento
- **Trabajo de Core**: Requiere activación coordinada

## Errores Comunes a Evitar

1. **Sobre-extender** - No fuerces la espalda baja
2. **Olvidar glúteos** - Activa antes de levantar
3. **Contener respiración** - Respira con el mecimiento
4. **Mucha velocidad** - Controla el movimiento de mecimiento

## Consejos para el Éxito

- Comienza con mecimientos muy pequeños
- Activa tus glúteos antes de levantar
- Usa tus manos para atrapar y controlar
